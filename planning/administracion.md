# Fase 6: Administración y Permisos Configurables

## 1. Gestión de Administradores de Evento

### Invitación

* Invitación por correo electrónico.
* El admin invitado tiene **30 minutos** para aceptar la invitación.
* Búsqueda de usuarios por correo para invitar.

### Remoción

* El organizador puede remover admin en cualquier momento.
* **Notificación**: el admin recibe notificación cuando es removido del evento.

### Límites

* No hay restricción de cantidad de admins por evento.

## 2. Panel de Administración de Plataforma

### Acceso

* El admin de plataforma accede a un **panel separado** para gestionar:
  * Usuarios del sistema
  * Categorías globales
  * Eventos (visibilidad completa)

### Diferenciación

* Organizadores y admins de evento acceden a la **misma app** con permisos restringidos a sus eventos.
* El admin de plataforma tiene su propio panel con funcionalidades exclusivas.

## 3. Permisos Configurables

### Concepto

Los permisos de cada rol son **configurables desde un panel administrativo**, permitiendo:

* Activar/desactivar permisos específicos para Admin de Evento.
* Personalizar qué acciones puede realizar cada rol.
* Adaptar el sistema a diferentes necesidades organizativas.

### Permisos Configurables para Admin de Evento

| Permiso | Default | Configurable |
|---------|---------|--------------|
| Editar info del evento | ❌ | ✅ |
| Crear/editar/eliminar torneos | ✅ | ✅ |
| Gestionar participantes en torneos | ✅ | ✅ |
| Configurar fase de grupos | ✅ | ✅ |
| Gestionar llave eliminatoria | ✅ | ✅ |
| Gestionar establecimientos/canchas | ❌ | ✅ |
| Invitar admins de evento | ❌ | ❌ |

## 4. Organización del Evento

### Reglas

* Un organizador **no puede transferir** su rol a otro usuario.
* Siempre hay un **único organizador** por evento.
* El organizador es designado por el admin de plataforma al crear el evento.
